أنا لدي Dashboard جاهز لمنصة تعليمية اسمها "مستندك الذكي" مبنية بأسلوب RTL عربي باستخدام React + Vite + TailwindCSS.

التصميم الحالي يحتوي على:
- Sidebar
- Topbar
- Upload section
- Stats cards
- Recent documents
- Quick actions

أريد منك الآن تطوير النظام ليصبح منتج ذكي متكامل، وليس مجرد Dashboard.

⚠️ مهم:
لا تعيد بناء المشروع من الصفر.
قم فقط بإضافة وتحسين الأجزاء الناقصة ورفع مستوى الذكاء والتجربة.

-----------------------------------

🎯 المطلوب (النواقص التي يجب إضافتها):

1. Chat with Document (أهم جزء)
- بناء صفحة Chat كاملة
- تصميم chat bubbles حديث
- دعم المحادثة مع الملف
- إضافة خيارات:
  - Basic explanation
  - Advanced explanation
  - Summary
- إضافة quick prompts:
  - لخص الملف
  - اشرح لي هذا الجزء
  - أعطني أهم النقاط
  - أنشئ اختبار
- عرض مصادر الإجابة (اسم الملف + الصفحة)
- loading state + typing animation

-----------------------------------

2. Quiz System
- صفحة توليد اختبار
- خيارات:

  - عدد الأسئلة
- عرض الأسئلة بطريقة cards
- زر تصحيح
- عرض النتيجة
- شرح الإجابات
- إعادة المحاولة

-----------------------------------

3. Smart Actions داخل Dashboard
أضف أزرار ذكية واضحة:
- 🎯 لخص الملف

- 📝 أنشئ اختبار
